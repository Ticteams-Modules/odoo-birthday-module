🎂 User Birthday

Never forget another teammate's birthday again.

Odoo 18.0 LGPL-3 By TICTeams

What does it do?

User Birthday adds a Birthday field to internal users and brings everything you need to celebrate your team: sorted lists, filters, kanban cards, and real-time popup notifications when somebody is celebrating today.

  • Adds a Birthday date field on every internal user.
  • New Birthdays menu sorted by upcoming celebration.
  • Filters: today, this week, within 30 days, missing birthday.
  • Group by birthday month.
  • Computed fields: next birthday, days left, age.
  • Daily scheduled action that automatically notifies the team.
  • Real-time popup notification (bus) for every connected user.
  • Manual Send notification now button on the user form.

Key features

📅 Birthday field

A simple date field on the user form, with validation (no future dates) and full chatter tracking.

📋 Sorted list

A dedicated list view sorting users by their next upcoming birthday with colour decorations.

🗂️ Filters & groups

Filter by today / this week / within 30 days, or group by birthday month.

🃏 Kanban view

A clean kanban board that visually highlights the user celebrating today.

🔔 Popup notifications

A daily cron pushes a real-time popup notification (bus) to every internal user, sticky until dismissed.

🛠️ Configurable cron

The schedule can be tweaked under Settings → Technical → Scheduled Actions.

How to use

  1. Install the module from Apps.
  2. Open any user and set their Birthday.
  3. Open the Birthdays top-level menu to see everyone sorted by upcoming celebration.
  4. The cron User Birthday: Daily Notification runs every day and pops up a notification when somebody is celebrating their birthday today.
